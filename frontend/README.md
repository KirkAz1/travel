# 游迹全国 · 前端

Vue 3 + Vite + Element Plus 打造的旅行内容平台前端，提供热门景点浏览、精选游记阅读、用户登录注册、收藏点赞与评论等交互体验。

## 快速开始

```bash
cd frontend
npm install
npm run dev
```

默认后端运行在 `http://localhost:3000`。如需修改，可在根目录创建 `.env` 文件：

```
VITE_API_BASE_URL=http://localhost:4000/api
```

## 可用脚本

| 命令 | 说明 |
| ---- | ---- |
| `npm run dev` | 本地开发，默认端口 5173 |
| `npm run build` | 生产构建 |
| `npm run preview` | 预览构建产物 |

## 功能概览

- **首页**：搜索筛选景点、查看热门景点卡片、浏览精选游记
- **认证**：登录 / 注册表单，成功后自动存储 token 与用户信息
- **游记详情**：展示图文内容、点赞收藏、评论发布与删除
- **收藏中心**：查看并管理收藏的景点/游记
- **全局导航**：动态显示登录状态、快速跳转收藏或退出

## 目录结构

```
src/
├─ api/           # axios 实例 & 业务接口
├─ assets/        # 全局样式
├─ components/    # 通用组件（导航等）
├─ router/        # 路由配置及守卫
├─ stores/        # 轻量级状态（auth）
└─ views/         # 页面视图
```

确保后端已启动且数据库就绪即可体验完整功能。若调用接口失败，请检查 `.env` 配置与后端服务状态。》
